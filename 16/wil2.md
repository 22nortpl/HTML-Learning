자바스크립트 자료형
  원시 유형
    숫자형 : 숫자로만
      정수와 실수로 나눔
    BigInt : 아주 큰 정수
    문자열 : 따옴표 가족과 함께 묶음
      특수 기호 표시
      \ddd : 8진수
      \xddd : 16진수
      \\ : 백슬래시
      \' : 작은따옴표 문자
      \" : 큰따옴표 문자
      \b : 백스페이스 문자
      \n : 줄바꿈
      \t : 탭
    논리형 : true or false
  복합 유형
    배열 : 하나의 변수에 값 여러개
    객체 : 함수와 속성 함께 포함
  특수 유형
    undefined : 자료형 지정 X
    null : 값이 유효 X
