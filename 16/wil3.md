연산자 종류
  산술 연산자
    + : 두 피연산자 덧셈
    - : 뒤 피연산자 뺄셈
    * : 곱셈
    / : 뒤 피연산자 나눔
    % : 나눈 나머지 구함
    ++ : 피연산자 1증가
    -- : 피연산자 1감소
  할당 연산자
    = : 연산자 오른쪽의 값 할당
    += : 더한 값 할당
    -= : 뺀 값 할당
    *, /, % 도 원래 연산자에 할당 원리만 적용!
  비교 연산자
    == : 피연산자 값만 같으면 true
    === : 자료형까지 같으면 true
    != : 값이 같지 않으면 true
    !== : 값 혹은 자료형 같지 않으면 true
    < : 왼쪽이 오른쪽보다 작으면 true
    <= : 왼쪽이 오른쪽 이하면 true
    > : 왼쪽이 오른쪽 초과면 true
    >= : 왼쪽이 오른쪽 이상이면 true
  논리 연산자
    || : OR
    && : AND
    ! : NOT
