DOM에서 함수 연결이 가능?

event 객체 사용시 가능!
  프로퍼티
    altKey : alt 눌린 여부
    button : 마우스에서 누른 버튼의 키값 반환
    charCode : 어떤 키 눌렀는지 유니코드로
    clientX : 발생한 가로 위치
    clientY : 발생한 세로 위치
    ctrlKey : ctrl 눌린 여부
    pageX : 현재 문서 기준 발생한 가로 위치
    pageY : 현재 문서 기준 발생한 세로 위치
    screenX : 현재 화면 기준 발생한 가로 위치
    screenY : 현재 화면 기준 발생한 세로 위치
    shiftKey : shift 눌린 여부
    target : 최초 발생 대상 반환
    timeStamp : 발생 시간 반환
    type : 이벤트 이름 반환
    which : 이벤트 발생 시 키의 유니코드 값 반환
  메소드
    preventDefault() : 이벤트 취소할 수 있는 경우 취소
