Array 객체 : 배열을 다룸
  메서드
  concat : 배열 요소 합침
  every : 모든 요수 참인지 파악
  filter : 필터링 함수에 대해 참인 요소로 새로운 배열
  forEach : 배열의 모든 요소에 주어진 함수 실행
  indexOf : 일치하는 값이 있는 배열 요소의 첫 인덱스
  join : 배열 요소를 문자열로, 구분자 지정 가능
  push : 끝에 새로운 요소 추가 후 새로운 length 반환
  unshift : 시작 부분에 새로운 요소 추가
  pop : 마지막 요소를 결과로 변환
  shift : 첫 번째 요소를 결과로 변환
  splice : 요소 추가 혹은 삭제
  slice : 특정 부분만 잘라냄
  reverse : 배치 순서를 역으로
  sort : 지정 조건에 따라 정렬
  toString : 지정한 부분 문자열로 반환

Date 객체 : 날짜 다룸
날짜 시간 정보 입수
  getFullYear() : 연도 4자리
  getMonth() : 달을 0~11로
  getDate() : 일을 1~31로
  getDay() : 요일을 0~6으로, 0이 Sunday
  getTime() : 시간을 밀리초로 표시
  getHours() : 시를 0~23 사이로
  getMinutes() : 분을 0~59 사이로
  getSeconds() : 초를 0~59 사이로
  getMilliseconds() : 밀리초를 0~999 사이로
날짜 시간 설정
  setFullYear() : 연도 4자리
  setMonth() : 달을 0~11로
  setDate() : 일을 1~31로
  setTime() : 시간을 밀리초로 표시
  setHours() : 시를 0~23 사이로
  setMinutes() : 분을 0~59 사이로
  setSeconds() : 초를 0~59 사이로
  setMilliseconds() : 밀리초를 0~999 사이로
형식 바꾸기
  toLocaleString() : 현지시간으로 표시
  toString() : 문자열로 표시

Math 객체 : 수학계산과 관련된 메소드
정해진 값(프로퍼티)
  E = e
  PI = 파이
  SQRT2 = 2의 제곱근
  SQRT1_2 = 1/2의 제곱근
  LN2 = ln2
  LN10 = ln10
  LOG2E = log_2_e
  LOG10E = loge
메서드
  abs() : 절댓값
  acos() : arccos
  asin() : arcsin
  atan(), atan2() : arctan
  ceil() : 소수점 이하 올림
  cos() : cos
  exp() : 지수함수
  floor() : 소수점 이하 버림
  log() : log
  max() : 최댓값
  min() : 최솟값
  pow() : 지숫값
  random() : 0~1 사이 무작위 수
  round() : 소수점 이하 반올림
  sin() : sin
  sqrt() : 제곱근
  tan() : tan
