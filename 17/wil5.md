마우스 이벤트
  click : 클릭 1회
  dbclick : 클릭 2회
  mousedown : 마우스 버튼 누름
  mousemove : 마우스 포인터 움직임
  mouseover : 요소 위로
  mouseout : 요소 벗어남
  mouseup : 손을 뗄 때
키보드 이벤트
  keydown : 키 누르는 동안
  keypress : 키 누르는 순간
  keyup : 키에서 손 뗄 때
문서 로딩 이벤트
  abort : 불러오기 중간에 멈춤
  error : 정확히 로딩 안 됨
  load : 로딩 완료
  resize : 크기 바뀔 때
  scroll : 화면 스크롤
  unload : 벗어날 때
폼 이벤트
  blur : 포커스 벗어날 때
  change : 체크 상태 등 변경 <input>, <select>, <textarea> 사용
  focus : 포커스가 맞춰지면 <label>, <select>, <textarea>, <button> 사용
  reset : 폼 리셋 될 때
  submit : submit 버튼 누를 때
