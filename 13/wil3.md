가상 클래스 : 사용자 동작에 반응
  1. 방문하지 않은 링크에 스타일 적용, :link 가상 클래스 선택자
  2. 방문한 링크에 스타일 적용, :visited 가상 클래스 선택자
  3. 특정 요소에 마우스 포인터를 올려놓으면 스타일 적용, :hover 가상 클래스 선택자
  4. 웹 요소를 활성화(클릭)했을 때 스타일 적용, :active 가상 클래스 선택자
  5. 웹 요소에 초점이 맞춰졌을 때 스타일 적용, :focus 가상 클래스 선택자

요소 상태에 따른 가상 클래스
  1. 앵커 대상에 스타일 적용, :target 가상 클래스 선택자
  2. 요소의 사용 여부에 따라 스타일 적용, :enabled, :disabled 가상 클래스 선택자
  3. 선택한 항목의 스타일 적용, :checked 가상 클래스 선택자

문서 구조 사용하는 가상 클래스
  :first-child : 첫 번째 자식 요소 선택
  :last-child : 마지막 자식 요소 선택
  :only-child : 자식 요소 하나일 때 선택
  :nth-child(숫자 또는 수식) : n번째 자식 요소 선택
  A:nth-of-type(숫자 또는 수식) : A요소를 기준 n번째 자식 요소 선택

특정요소 제외하고 스타일 적용하는 :not 가상 클래스 선택자

:is 가상 클래스 :  최근 추가, 같은 스타일 여러 요소에 반복 적용
A:has(B) 는 괄호 안의 요소를 가지고 있는지 확인
