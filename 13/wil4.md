:root 가상 클래스 : 최상위 요소 선택
CSS 에서 변수를 사용하기 위함.
하이픈 2개 입력 필수!

var() : CSS 변수 불러오기 위해 사용
calc() : 계산식 으로 값 설정

min(), max() : 괄호 안의 값 중 최솟값, 최댓값 적용
clamp() : 값 범위 제한(최솟값, 최적값, 최댓값)

filter 속성과 함수
  blur : 흐림 정도
  brightness : 밝기 정도
  contrast : 대비 정도
  dropshadow : 이미지 윤곽 따라 그림자 적용(가로 오프셋, 세로 오프셋, 흐림 정도, 번짐 정도, 그림자 색)
  grayscale : 이미지를 회색조로 변경해서 표시
  invert : 반전 정도
  sepia : 세피아 강도
  opacity : 불투명도
  hue-rotate : 각도
  saturate : 채도
