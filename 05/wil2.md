<input> 태그 : 입력창 생성
  text : 한 줄짜리
  password : 비밀번호
    size : 텍스트와 비밀번호 필드의 길이를 지정
    value : 텍스트 필드 요소가 화면에 표시될 때 텍스트 필드 부분에 보여 주는 내용
    maxlength : 최대 문자 수 지정
  search : 검색
  url : 링크
  email : 이메일 주소
  tel : 전화번호
  checkbox : 체크박스 (2개 이상 선택)
  radio : 1개만 선택
    value : 넘겨줄 값 지정(영문 혹은 숫자)
    checked : 기본적 선택된 상태로 만들고 싶을 때
    name : 같은 이름을 사용해 묶는 속성
  number : 숫자 조절
  range : 범위 조절 막대
    min : 최솟값
    max : 최댓값
    step : 간격
    value : 초깃값
  date : 연, 월, 일
  month : 연, 월
  week : 연, 주
  time : 시, 분, 초, 분할 초
  datatime-local : 지역 기준 날짜 와 시간
    min : 최소 날짜
    max : 최대 날짜
    step : 화살표 클릭 시 증감시킬 크기
    value : 기본적으로 표시할 날짜 지정
  submit : 전송 버튼
  reset : 리셋 버튼
  image : submit 대신 사용 이미지 버튼
  button : dlfqks qjxms
  file : 파일 첨부 버튼
  hidden : 서버로 넘기는 값이 있는 필드 생성
