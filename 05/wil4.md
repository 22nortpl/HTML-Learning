<textarea> 태그 : 여러 줄을 입력하는 태그
  cols : 텍스트 영역의 가로 너비를 문자 단위로
  rows : 텍스트 영역의 세로 길이를 줄 단위로 지정
<select> 태그 : 드롭다운 목록 생성 (시작과 끝 표시)
  size : 화면에 표시할 드롭다운 항목 개수 지정
  multiple : 드롭다운 목록에서 둘 이상의 항목 선택시 사용
<option> 태그 : 드롭다운에서 원하는 항목 추가
  value : 해당 항목 선택 시 서버로 넘길 값 지정
  selected : 드롭다운 메뉴 삽입 시 기본 선택으로 보이는 항목 지정
<button> : 버튼 활성화 시 어떤 동작할지 지정
  submit : 폼을 서버로 전송
  reset : 폼에 입력한 내용 초기화
  button : 버튼 형태만 생성, 기능 없음
