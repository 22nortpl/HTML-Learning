<form> 태그 : 정보를 보내는 폼(form)을 만드는 태그
  method 속성 : 사용자가 입력한 내용을 서버 쪽 프로그램으로 어떻게 넘겨줄지 결정
  name 속성 : 자바스크립트로폼을 제어할 때 사용할 폼의 이름을 지정
  action 속성 : 폼 태그 안의 내용을 처리해줄 서버 프로그램을 지정
  target 속성 : action 속성에서 지정한 스크립트
<fieldset> 태그 : 하나의 폼 안에서 여러 구역
<legend> 태그 : 제목으로 붙여지는 태그
<label> 태그 : 레이블을 붙일 때 사용
<input> 태그 : 입력창 생성
