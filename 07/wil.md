font family:<글꼴 이름>
여러 개를 넣을 경우, 왼쪽 순서대로 우선 적용

웹 폰트나 font awesome 에서 폰트를 import 도 가능!

텍스트 스타일
color : <색상>
16진수로 표현, 혹은 hsl(a)로도 표현
이외에, 이름, rgb로도 표현 가능

텍스트 정렬
text-align 사용
종류
  start : 시작위치에 맞춤 (기본값)
  end : 끝 위치에 맞춤
  left : 왼쪽에 맞춤
  right : 오른쪽에 맞춤
  center : 가운데에 맞춤
  justify : 양쪽에 맞춤
  match-parent : 부모 요소를 따라 정렬

line-height : 줄 높이 조정 속성
text-decoration : 밑줄 표시
text-shadow : 그림자 표과 추가
  가로 거리 : 필수 속성

목록 스타일
list-style-type 내의 속성
  disc : 채워진 원
  circle : 빈 원
  square : 채워진 사각형
  decimal : 10진수(1부터)
  decimal-leading-zero : 0붙은 10진수(01, 02)
  lower-roman : i, ii, iii
  upper-roman : I, II, III
  lower-alpha/lower-latin : a,b,c...
  upper-alpha/upper-latin : A,B,C...
  none : 아무것도 없음
list-style-image : 이미지를 사용
list-style-position : 글을 들여쓸지 결정
  inside : 기본 위치보다 안에
  outside : 기본값

이 모두를 통합해서 list-style로 표현 가능!

caption-side : 제목 위치 지정 (top 혹은 bottom)
border 속성 : 표 바깥 테두리와 셀 테두리 지정
border-collapse : 표와 셀의 테두리를 합쳐 하나로 표시 (기본값은 separate)
