animation : 웹 요소에 애니메이션 추가
  @keyframes : 애니메이션 바뀌는 지점 지정
    animation-name : 이름 지정
    animation-duration : 지속 시간 지정
  animation-delay : 지연 시간 지정
  animation-direction : 처음 혹은 역방향 지정
    normal : 기본값, from 에서 to로 지정
    reverse : 반대
    alternate : 홀수는 normal, 짝수는 reverse
    alternate-reverse : alternate 와 반대
  animation-duration : 실행시간 지정
  animation-iteration-count : 반복횟수 지정
  animation-name : @keyframes로 설정한 애니메이션 지정
  animation-timing-function : 키프레임의 전환 형태 지정
  animation : 한번에 모두 지정
