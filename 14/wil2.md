transition : 시간에 따라 변하는 속성
  transition-property : 대상 지정
    all : 모두 변함
    none : 아무도 안 변함
    속성 이름 : 그 대상 지정
  transition-duration : 진행 시간 지정
  transition-timing-function : 속도 곡선 지정
    ease : 계수 음수인 이차곡선, 기본값
    linear : 일정한 가속도
    ease-in : 느리게 시작
    ease-out : 느리게 끝
    ease-in-out : 느리게 시작, 느리게 끝
    cubic-bezier(n, n, n, n) : 베지에 함수 정의. 0<=n<=1
  transition-delay : 지연시간 지정
  transition : 한번에 모두
