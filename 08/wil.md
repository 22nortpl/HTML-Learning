블록 레벨 요소와 인라인 레벨 요소:
  블록 레벨 요소 : h1, div, p 처럼 혼자 한 줄을 차지
  인라인 레벨 요소 : span, img, strong 처럼 한 줄에 여러개 표시가 가능

콘텐츠 영역의 크기를 지정하는 width, height 속성 존재, 숫자 지정, 백분율 지정 혹은 자동으로

box-sizing으로 박스 모델의 크기 계산
border-box : 패딩과 테두리를 포함 너빗값 지정
content-box : 기본값, 콘텐츠 영역만

box-shadow : 박스 모델에 그림자 효과
  수평 거리 : 그림자의 가로 떨어진 정도
  수직 거리 : 그림자의 세로 떨어진 정도
  흐림 정도 : 숫자 낮을수록 진함
  번짐 정도 : 숫자 낮을수록 그림자 작음
  색상 : 색 지정
  inset :  그림자 방향 반대
