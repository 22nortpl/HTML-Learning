테두리 스타일 지정
border-style 속성:
  none : 테두리 X
  hidden : 테두리 숨김
  solid : 테두리 실선
  dotted : 테두리 점선
  dashed : 테두리를 짧은 직선으로
  double : 테두리를 이중선으로
  groove : 창에 조각한 듯
  inset : separate 일 때 창에 박힌 듯 표시, collapse 는 inset과 동일.
  outset : separate 일 때 창에서 튀어나온 듯 표시, collapse 는 ridge 와 동일.
  ridge : 튀어나온 듯 표시

border-width 속성:
  크기 속성 지님(thick, medium, thin)

border-color 속성: 원하는 색깔
border-radius 속성 : 네 군데 꼭짓점 둥글게, 혹은 원형으로
